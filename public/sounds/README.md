# Звуковые файлы для чата AI-консультанта

## Требуемые файлы

Для работы звукового сопровождения чата необходимо добавить два аудио-файла:

1. **send.wav** (или send.mp3) - короткий мягкий звук при отправке сообщения пользователем
   - Рекомендуемая длительность: 0.1-0.3 секунды
   - Тип звука: мягкий "клик" или "тик"
   - Громкость: средняя
   - Поддерживаемые форматы: WAV, MP3, OGG

2. **reply.wav** (или reply.mp3) - приятный звук при получении ответа AI-консультанта
   - Рекомендуемая длительность: 0.3-0.5 секунды
   - Тип звука: мягкий "дзинь" или "чек"
   - Громкость: средняя
   - Поддерживаемые форматы: WAV, MP3, OGG

## Где найти бесплатные звуки

- [Freesound.org](https://freesound.org/) - бесплатная библиотека звуков
- [Zapsplat](https://www.zapsplat.com/) - бесплатные звуковые эффекты
- [Mixkit](https://mixkit.co/free-sound-effects/) - бесплатные звуки

## Примеры поиска

- Для send.mp3: ищите "click", "tick", "notification", "ui click"
- Для reply.mp3: ищите "ding", "chime", "notification success", "bell"

## Формат

Файлы должны иметь названия точно:
- `send.wav` (или `send.mp3`, `send.ogg`)
- `reply.wav` (или `reply.mp3`, `reply.ogg`)

**Поддерживаемые форматы:** WAV, MP3, OGG (все форматы поддерживаются браузерами через HTML5 Audio API)

**Рекомендация:** WAV формат обеспечивает лучшее качество звука, но файлы могут быть больше по размеру. MP3 более компактный.

Разместите файлы в этой папке (`public/sounds/`).
